## ✨ Features

- 📄 **Document Upload & Download**
    - Upload client documents to Amazon S3
    - Download files using bucket and object key

- 📊 **Portfolio Management**
    - Create portfolios for clients
    - Manage portfolio-related operations

- 💹 **Trade Order Processing**
    - Submit BUY orders to the Trade Service
    - Event-driven architecture using Kafka

- ☁️ **AWS Integration**
    - Amazon S3 for document storage
    - Amazon RDS (MySQL) for persistence
    - AWS Secrets Manager support

- 🚀 **Asynchronous Messaging**
    - Kafka-based event publishing
    - Decoupled service communication

- 🔒 **Secure Configuration**
    - Environment variables support
    - IAM role integration
    - Secrets Manager integration

- 📚 **API Documentation**
    - Swagger/OpenAPI UI support  http://localhost:8081/swagger-ui/index.html
    - Interactive API testing
